# IPL Data Analysis (2008–2025)

## Project Overview
This project analyzes Indian Premier League (IPL) data from 2008 to 2025 using data visualization techniques. The objective is to extract meaningful insights about team performance, player statistics, and match trends through an interactive dashboard.

## Features
- Year-wise IPL analysis
- Team performance comparison
- Player performance statistics (runs, wickets, etc.)
- Match outcome trends
- Venue-based insights
- Toss impact analysis
- Win and loss patterns

## Tools and Technologies
- Power BI (Dashboard and visualization)
- Python (optional for preprocessing)
- Pandas and NumPy (optional)
- Excel / CSV datasets


## Key Insights
- Identification of top-performing teams across seasons
- Analysis of players with highest runs and wickets
- Impact of toss decisions on match results
- Performance variation across different venues
- Trends in scoring patterns over the years
