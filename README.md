# IPL Analysis Dashboard (2008–2025)

## Project Overview
This project presents an interactive Power BI dashboard analyzing Indian Premier League (IPL) data from 2008 to 2025. It provides insights into team performance, player achievements, and match statistics using visually rich and user-friendly analytics.

## Dashboard Highlights
- Season selector to filter data dynamically
- Champion and Runner-up overview for selected season
- Total statistics including:
  - Total 6s
  - Total 4s
  - Total matches
  - Total teams
  - Total centuries
- Orange Cap analysis (top run scorer)
- Purple Cap analysis (top wicket taker)
- Team-wise comparison of:
  - Total 6s
  - Total 4s
  - Total runs
- Additional insights:
  - Most 4s in a season
  - Most 6s in a season

## Key Insights
- Royal Challengers Bangalore shows highest total runs in overall comparison
- Strong correlation between top players and team success
- Boundary-heavy teams tend to have better match outcomes
- Individual player performances significantly impact seasonal results

## Tools and Technologies
- Power BI (Dashboard development and visualization)
- Data preprocessing (Excel / CSV datasets)
